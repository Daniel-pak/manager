# Manager

Manager is an application that allows you to easily syndicate your employee's work schedules via text.

  - `npm install`
  - `react-native run-[operating system]`

### Plugins

Manager is currently extended with the following plugins.

| ------ |
| React-Redux |
| Redux |
| Firebase |
